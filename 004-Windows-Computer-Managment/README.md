# 004 – Windows Computer Management

## Overview

This project demonstrates how Windows Computer Management is used to inspect and administer a workstation before deployment within a simulated enterprise environment.

The project covers reviewing administrative tools, local users, shared folders, hardware devices, Windows services, and documenting the workstation's overall health.

---

## Scenario

A newly deployed employee workstation must be inspected before being issued to a staff member.

The IT Support technician uses Computer Management to verify that the workstation is properly configured and operating normally.

---

## Objectives

- Access Computer Management
- Review System Tools
- Review Local Users and Groups
- Review Shared Folders
- Review Device Manager
- Review Windows Services
- Document findings
